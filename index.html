Black Ops 4

Call of Duty: Black Ops 4 is as impressive as it is anomalous. The game contains very little that hasn't been done first by others, but it approaches those features in a way that's so polished and unique that it doesn't really matter if they're particularly original

Gameplay

The solo content is indeed the longest of the franchise, but the solo modes are broken up into single missions and not a unified campaign. So put an asterisk on this title if that doesn't seem like a qualifying single-player experience.

Difficulty

Casual: Player starts with 250 health (5 hits). Normal: Player starts with 200 health (4 hits). Hardcore: Player starts with 150 health. (3 hits).

Graphics

The Graphic Content Filter (or Disturbing Content Filter) is a feature which significantly reduces mature content such as blood, violence and strong language.
